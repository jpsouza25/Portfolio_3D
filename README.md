## Tecnologias

---

## Instruções para rodar localmente

# Clone o repositório
git clone https://github.com/seu_usuario/portfolio-joaopedro.git

# Entre na pasta do projeto
cd portfolio-joaopedro

# Instale as dependências
npm install

# Rode o servidor de desenvolvimento
npm run dev
